A professional portfolio website showcasing my profile, skills, projects, and achievements.
It is designed to create a strong online presence with a clean and responsive layout.

### **Task 2: E-commerce Product Page**

A modern product landing page for **Aura Pro Wireless Headphones**.
This project highlights product details, pricing, features, and a visually appealing design for an online shopping experience.

### **Task 3: Simple Navigation Menu**

A responsive and user-friendly navigation menu built using HTML and CSS.
It demonstrates clean website navigation structure and basic UI design principles.

### **Task 4: Interactive Contact Form**

A contact form with input validation and interactive design.
This task focuses on creating a functional form layout for collecting user details effectively.

### **Task 5: Modern Blogging Platform**

A modern blog-style webpage layout designed to display articles in an organized and attractive format.
It emphasizes readability, clean structure, and responsive design.

### **Task 6: Interactive Budget Tracker**

A budget tracking web application that allows users to add income and expenses, monitor transactions, and manage their finances interactively.
